## 使い方

* WASD：
     移動
* 左クリック：
     射撃
* 右クリック：
     ズーム
* C：
     射撃モード切替
* Space：
     ジャンプ
* L-Shift：
     ダッシュ
* Esc：
     マウスカーソルの表示（ウィンドウの×を押して終了）
* Alt+F4：
    終了
